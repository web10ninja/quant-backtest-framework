## 📋 About quant-backtest-framework

quant-backtest-framework is a tool to help you test trading strategies using historical data. It lets you check how your strategies might have worked in the past. The framework includes several testing methods, like walk-forward optimization, Monte Carlo validation, and stress testing. These help you see how your plan performs in different scenarios.

You don’t need to be a programmer to use it. This guide walks you through everything you need to know to download and run the software on your Windows computer.

### Key Features

- Test your trading ideas against past market data  
- Use walk-forward optimization to check if your strategy adapts over time  
- Monte Carlo testing to measure robustness across random simulations  
- Stress testing for unusual or extreme market events  
- Clear signals for GO or NO-GO decisions on live deployment  
- Works with common data formats like CSV files  
- Runs without complex setup or programming skills

## 🐍 Setting up Python (If Needed)

The framework requires Python 3.7 or newer to run. If you are unsure if Python is installed, follow these steps.

### Check if Python is installed:

1. Press `Win + R` to open the Run box.  
2. Type `cmd` and press Enter.  
3. In the black command window, type `python --version` and press Enter.  
4. If it shows a version like "Python 3.x.x," you already have it installed.  

### How to install Python:

1. Go to the official Python website: https://www.python.org/downloads/windows/  
2. Download the latest version labeled "Windows installer."  
3. Open the downloaded file and check the box that says "Add Python 3.x to PATH" at the bottom.  
4. Click “Install Now.”  
5. Wait for the installation to complete, then close the window.
